# AfriClim_AI_MVP
AfriClim AI MVP est une plateforme d’évaluation des risques climatiques basée sur l’intelligence artificielle. Elle analyse des données météo en temps réel pour générer des scores de risque localisés, estimer l’impact économique potentiel et proposer des recommandations concrètes afin de renforcer la résilience des villes africaines.
